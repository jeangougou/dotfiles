# Suggeted setup

## Install Bash-it

Clone locally and enable

```
git clone --depth=1 https://github.com/Bash-it/bash-it.git ~/.bash_it
~/.bash_it/install.sh
```

## Run local installation

and hope for the best

```
sudo apt update
sudo apt install -y ansible
./__init.sh
```
